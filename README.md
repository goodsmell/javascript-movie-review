# javascript-movie-review

FE 레벨1 영화 리뷰 미션

# 요구사항 목록

## 1. 검색

- 검색어를 입력받아 검색한다.
- 검색어에 해당하는 영화를 10개 가져온다.
- 더보기를 누르면 10개를 추가로 가져온다.
- 해당하는 모든 영화를 가져왔을 때 더 보기 버튼을 숨긴다.
- 검색어에 해당하는 영화가 없을 경우, "검색 결과가 없습니다"
- 검색 시 스켈레톤 UI를 표시한다.

## 2. 영화 목록

- 영화 20개를 가져온다.
- 더보기를 누르면 20개를 추가로 가져온다.
- 모든 영화를 가져왔을 때 더 보기 버튼을 숨긴다.
- 검색 시 스켈레톤 UI를 표시한다.

## 배너

- 가장 인기 있는 영화의 정보를 띄운다.

# 테스트 명세

## 단위 테스트

1. 인기 있는 영화 목록을 가져온다.
2. 검색어에 맞는 영화만 가져온다.
